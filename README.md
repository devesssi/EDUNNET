# 🎓 EDUNNET

🌟 An educational repository to learn, explore, and build projects in **Artificial Intelligence, Machine Learning, Deep Learning, Computer Vision, and NLP** — through hands-on Colab notebooks & real datasets.

This repo serves as a **step-by-step learning journey**, covering both foundational concepts and applied projects.

---

## 📁 Repository Structure

| 📄 File Name                                      | 📖 Description                                                                                                                                                                        |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `🌪️_Predicting_Regional_ClimateRiskLevels.ipynb` | 🌪️ A complete ML pipeline predicting regional climate risk levels using socio-economic & environmental data. Includes cleaning, feature engineering, model training, and evaluation. |
| `ANN_model.ipynb`                                 | 🧠 Demonstrates how to build & train an Artificial Neural Network (ANN) using Keras/TensorFlow — explaining forward/backward propagation, activation functions, and weights.          |
| `DAY04edunet.ipynb`                               | 📅 Day 4 notebook — covers intermediate concepts in ML/DL. Could include model evaluation techniques, optimization, or data visualization.                                            |
| `Data_analysis(day2).ipynb`                       | 📊 Day 2 notebook — focused on exploratory data analysis (EDA): cleaning, visualizing, and summarizing datasets to understand data before modeling.                                   |
| `EDUNETDAY03_part_01.ipynb`                       | 🔷 Day 3 notebook (part 1) — likely covering the first part of ML model building or theoretical understanding of algorithms.                                                          |
| `Edunetday1.ipynb`                                | 🚀 Day 1 notebook — an introduction to machine learning & AI basics: what is AI, supervised vs unsupervised, and simple hands-on examples.                                            |
| `Edunetday2.ipynb`                                | 📊 Day 2 continuation — deeper data analysis, handling missing values, and preparing data for ML models.                                                                              |
| `logistic(work).ipynb`                            | 📈 Shows how to implement logistic regression from scratch or with scikit-learn for binary classification problems. Includes theory & practice.                                       |
| `model_buildingDAY03(part_2).ipynb`               | 🔷 Day 3 notebook (part 2) — continuation of Day 3, likely includes training and evaluating ML models or experimenting with hyperparameters.                                          |
| `README.md`                                       | 📜 This documentation file, explaining the project, file descriptions, how to run notebooks, and future roadmap.                                                                      |

## 🚀 How to Run

### 1️⃣ Clone the repository

```bash
git clone https://github.com/devesssi/EDUNNET.git
cd EDUNNET
```

### 2️⃣ Open Notebooks

You can run these notebooks directly in **Google Colab**:

* Go to [Colab](https://colab.research.google.com/), click **File > Open notebook > GitHub**, and paste the repo link.
* Or open the `.ipynb` files locally in Jupyter Notebook.
